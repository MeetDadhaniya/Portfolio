# MEET — Tactical Portfolio · Full Technical Handoff

A pixel-art / Valorant-HUD style personal portfolio. Single-page Next.js App Router site with dynamic per-project detail routes, smooth scroll via Lenis, animations via Framer Motion, sounds via Web Audio API.

---

## 1) STACK & DEPENDENCIES

**Framework**: Next.js 15 (App Router) on React 18, plain JavaScript (`.js` / `.jsx`, no TypeScript)
**Styling**: Tailwind CSS 3 + a small set of custom CSS utilities in `globals.css`
**Animation**: Framer Motion
**Smooth scroll**: Lenis
**Icons**: lucide-react (only used in `MatchSummary` for socials)
**Fonts**: Press Start 2P (pixel headings) + JetBrains Mono (HUD/body), both loaded via `next/font/google`
**Audio**: Browser-native Web Audio API (no library) — implemented in `/lib/sound.js`

### `app/page.js`

```jsx
'use client';
import { useEffect, useRef, useState } from 'react';
import Lenis from 'lenis';
import { AnimatePresence } from 'framer-motion';

import MainMenu from '@/components/MainMenu';
import MiniMap from '@/components/MiniMap';
import AgentBio from '@/components/AgentBio';
import AgentRoster from '@/components/AgentRoster';
import Loadout from '@/components/Loadout';
import MatchSummary from '@/components/MatchSummary';
import CrosshairCursor from '@/components/CrosshairCursor';
import KillfeedToast from '@/components/KillfeedToast';
import PingEffect from '@/components/PingEffect';
import SoundProvider from '@/components/SoundProvider';
import RoundTransition from '@/components/RoundTransition';

function App() {
  const [showMenu, setShowMenu] = useState(true);
  const [activeSection, setActiveSection] = useState('about');
  const lenisRef = useRef(null);

  useEffect(() => {
    if (showMenu) return;
    const lenis = new Lenis({
      lerp: 0.08,
      smoothWheel: true,
      wheelMultiplier: 1.0,
    });
    lenisRef.current = lenis;
    function raf(time) {
      lenis.raf(time);
      requestAnimationFrame(raf);
    }
    const id = requestAnimationFrame(raf);
    return () => {
      cancelAnimationFrame(id);
      lenis.destroy();
      lenisRef.current = null;
    };
  }, [showMenu]);

  useEffect(() => {
    if (showMenu) return;
    const sections = ['about','work','skills','contact'];
    const observers = [];
    sections.forEach((id) => {
      const el = document.getElementById(id);
      if (!el) return;
      const obs = new IntersectionObserver(
        (entries) => {
          entries.forEach((entry) => {
            if (entry.isIntersecting) setActiveSection(id);
          });
        },
        { threshold: 0.35 }
      );
      obs.observe(el);
      observers.push(obs);
    });
    return () => observers.forEach(o => o.disconnect());
  }, [showMenu]);

  const handleNavigate = (sectionId) => {
    const el = document.getElementById(sectionId);
    if (!el) return;
    if (lenisRef.current) {
      lenisRef.current.scrollTo(el, { offset: -20, duration: 1.2 });
    } else {
      el.scrollIntoView({ behavior: 'smooth' });
    }
  };

  return (
    <>
      <CrosshairCursor />
      <PingEffect />
      <SoundProvider />

      <AnimatePresence mode="wait">
        {showMenu && (
          <MainMenu key="menu" onPlay={() => setShowMenu(false)} />
        )}
      </AnimatePresence>

      {!showMenu && (
        <>
          <header className="fixed top-0 left-0 right-0 z-40 flex items-center justify-between px-4 md:px-6 py-3 bg-[#0d0d0d]/80 backdrop-blur border-b border-[var(--border)]">
            <div className="flex items-center gap-3">
              <button
                data-interactive
                onClick={() => setShowMenu(true)}
                className="font-pixel text-[10px] uppercase tracking-widest text-[var(--fg)] hover:text-[var(--accent)] transition"
              >
                ◀ MENU
              </button>
              <span className="text-[var(--muted)]">|</span>
              <span className="font-pixel text-[10px] uppercase tracking-widest text-[var(--accent)]">MEET</span>
              <span className="font-mono text-[10px] uppercase tracking-widest text-[var(--muted)] hidden sm:inline">// AGENT FILE</span>
            </div>
            <div className="hidden md:flex items-center gap-4 font-mono text-[10px] uppercase tracking-widest text-[var(--muted)]">
              <span>HP 100</span>
              <span>ARMOR 50</span>
              <span>CREDS 9000</span>
              <span className="text-[var(--accent)]">● LIVE</span>
            </div>
          </header>

          <main className="relative pt-16 scanlines">
            <RoundTransition label="ROUND 01 · AGENT BIO" code="// CHECKPOINT 01" sound="round">
              <AgentBio />
            </RoundTransition>
            <SectionDivider code="// CHECKPOINT 02" />
            <RoundTransition label="ROUND 02 · AGENT ROSTER" code="// CHECKPOINT 02" sound="match">
              <AgentRoster />
            </RoundTransition>
            <SectionDivider code="// CHECKPOINT 03" />
            <RoundTransition label="ROUND 03 · LOADOUT" code="// CHECKPOINT 03" sound="round">
              <Loadout />
            </RoundTransition>
            <SectionDivider code="// CHECKPOINT 04" />
            <RoundTransition label="ROUND 04 · VICTORY" code="// MATCH END" sound="match">
              <MatchSummary />
            </RoundTransition>

            <footer className="border-t border-[var(--border)] mt-12 px-6 py-8 text-center">
              <div className="font-mono text-[10px] uppercase tracking-widest text-[var(--muted)]">
                © MEET.GG &middot; BUILT WITH NEXT.JS &middot; PATCH NOTES 1.0.0 &middot; GG WP
              </div>
            </footer>
          </main>

          <MiniMap activeSection={activeSection} onNavigate={handleNavigate} />
          <KillfeedToast />
        </>
      )}
    </>
  );
}

function SectionDivider({ code }) {
  return (
    <div className="max-w-7xl mx-auto px-6 md:px-12 py-6 flex items-center gap-3">
      <span className="w-2 h-2 bg-[var(--accent)]" />
      <span className="font-mono text-[10px] uppercase tracking-[0.3em] text-[var(--muted)]">{code}</span>
      <span className="flex-1 h-px bg-[var(--border)]" />
    </div>
  );
}

export default App;
```

import { notFound } from 'next/navigation';
import { PROJECTS, getProjectBySlug } from '@/lib/projects';
import AgentDetailClient from './AgentDetailClient';

export function generateStaticParams() {
  return PROJECTS.map((p) => ({ slug: p.slug }));
}

export async function generateMetadata({ params }) {
  const { slug } = await params;
  const p = getProjectBySlug(slug);
  if (!p) return { title: 'AGENT NOT FOUND' };
  return {
    title: `${p.name} // AGENT FILE — MEET`,
    description: p.hook,
  };
}

export default async function AgentPage({ params }) {
  const { slug } = await params;
  const project = getProjectBySlug(slug);
  if (!project) return notFound();
  return <AgentDetailClient project={project} />;
}

'use client';
import { useEffect } from 'react';
import Sound from '@/lib/sound';

// Mounts global hover/click delegation so any button or [data-interactive]
// element gets a sound effect, without touching every component.
export default function SoundProvider() {
  useEffect(() => {
    if (typeof window === 'undefined') return;
    Sound.init();

    const isInteractive = (el) =>
      !!el?.closest?.('a, button, [role="button"], [data-interactive]');

    let lastHoverTarget = null;
    const onOver = (e) => {
      const target = e.target?.closest?.('a, button, [role="button"], [data-interactive]');
      if (!target || target === lastHoverTarget) return;
      lastHoverTarget = target;
      Sound.hover();
    };
    const onOut = (e) => {
      const target = e.target?.closest?.('a, button, [role="button"], [data-interactive]');
      if (target === lastHoverTarget) lastHoverTarget = null;
    };
    const onClick = (e) => {
      if (e.target?.closest?.('[data-no-click-sound]')) return;
      if (isInteractive(e.target)) Sound.click();
    };

    document.addEventListener('mouseover', onOver, true);
    document.addEventListener('mouseout', onOut, true);
    document.addEventListener('click', onClick, true);
    return () => {
      document.removeEventListener('mouseover', onOver, true);
      document.removeEventListener('mouseout', onOut, true);
      document.removeEventListener('click', onClick, true);
    };
  }, []);

  return null;
}

  return { id: Date.now() + Math.random(), startX, endX, y, speed, dir };
}

export default function BackgroundBirds() {
  const [birds, setBirds] = useState([]);
  const [kills, setKills] = useState(0);
  const [toasts, setToasts] = useState([]);
  const birdsRef = useRef(birds);
  birdsRef.current = birds;

  useEffect(() => {
    if (typeof window === 'undefined') return;
    let timer;
    const schedule = () => {
      const delay = SPAWN_MIN + Math.random() * (SPAWN_MAX - SPAWN_MIN);
      timer = setTimeout(() => {
        if (birdsRef.current.length < MAX_BIRDS) {
          const b = randomBird();
          if (b) setBirds((prev) => [...prev, b]);
        }
        schedule();
      }, delay);
    };
    const first = setTimeout(() => {
      const b = randomBird();
      if (b) setBirds((prev) => [...prev, b]);
      schedule();
    }, 1200);
    return () => { clearTimeout(timer); clearTimeout(first); };
  }, []);

  const handleDespawn = useCallback((id) => {
    setBirds((prev) => prev.filter((b) => b.id !== id));
  }, []);

  const handleKill = useCallback((x, y) => {
    setKills((k) => k + 1);
    try { Sound.headshot?.(); } catch (_) {}
    const id = Date.now() + Math.random();
    const label = Math.random() < 0.5 ? 'HEADSHOT' : 'TARGET ELIMINATED';
    setToasts((prev) => [...prev, { id, x, y, label }]);
    setTimeout(() => {
      setToasts((prev) => prev.filter((t) => t.id !== id));
    }, 900);
  }, []);

  return (
    <>
      <div className="absolute inset-0 z-0 overflow-hidden" aria-hidden>
        {birds.map((b) => (
          <Bird key={b.id} {...b} onDespawn={handleDespawn} onKill={handleKill} />

      <div className="pointer-events-none absolute inset-0 z-[1100]" aria-hidden>

              key={t.id}
              initial={{ opacity: 0, y: 0, scale: 0.8 }}
              animate={{ opacity: 1, y: -28, scale: 1 }}

              style={{ left: t.x, top: t.y, transform: 'translate(-50%, -50%)' }}
            >
              <span className="hud-border-accent bg-[#0d0d0d]/90 px-2 py-1 inline-block">
                {t.label}
              </span>

          ))}

      </div>

        BIRDS&nbsp;<span className="text-[var(--accent)]">{String(kills).padStart(2, '0')}</span>
      </div>
    </>
  );
}

function Bird({ id, startX, endX, y, speed, dir, onDespawn, onKill }) {
  const [pos, setPos] = useState({ x: startX, dy: 0, rot: 0, opacity: 1, frame: 0 });
  const [dead, setDead] = useState(false);
  const stateRef = useRef({ x: startX, deathT: 0, flapT: 0, dead: false });

  useEffect(() => {
    let raf;
    let last = performance.now();
    const tick = (now) => {
      const dt = Math.min(0.05, (now - last) / 1000);
      last = now;
      const s = stateRef.current;

      if (!s.dead) {
        s.x += dir * speed * dt;
        s.flapT += dt;
        let frame = pos.frame;
        if (s.flapT > 0.22) {
          frame = 1 - frame;
          s.flapT = 0;
        }
        const done = (dir > 0 && s.x > endX) || (dir < 0 && s.x < endX);
        if (done) { onDespawn(id); return; }
        setPos((p) => ({ ...p, x: s.x, frame }));
      } else {
        s.deathT += dt;
        const dy = 0.5 * 1600 * s.deathT * s.deathT;          // gravity (px/s^2)
        const rot = (dir > 0 ? 720 : -720) * (s.deathT / 0.8);
        const opacity = Math.max(0, 1 - s.deathT / 0.8);
        setPos((p) => ({ ...p, dy, rot, opacity }));
        if (s.deathT > 0.85) { onDespawn(id); return; }
      }
      raf = requestAnimationFrame(tick);
    };
    raf = requestAnimationFrame(tick);
    return () => cancelAnimationFrame(raf);
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, []);

  const handleClick = (e) => {
    if (stateRef.current.dead) return;
    e.stopPropagation();
    stateRef.current.dead = true;
    setDead(true);
    onKill(e.clientX, e.clientY);
  };

  return (
    <button
      type="button"
      onClick={handleClick}
      data-no-click-sound
      className="absolute p-1 bg-transparent border-0 outline-none"
      style={{
        left: pos.x,
        top: y + pos.dy,
        transform: `rotate(${pos.rot}deg)`,
        opacity: pos.opacity,
        cursor: 'none',
        willChange: 'transform, opacity, left, top',
      }}
      aria-label="shoot bird"
    >
      <BirdSprite frame={dead ? 'dead' : pos.frame} flip={dir < 0} />
    </button>
  );
}

// Tiny Web Audio FX synth. No external assets.
// Lazy-inits the AudioContext on first user gesture (browser policy).

let ctx = null;
let master = null;
let enabled = true;

function ensureCtx() {
  if (typeof window === 'undefined') return null;
  if (ctx) return ctx;
  try {
    const AC = window.AudioContext || window.webkitAudioContext;
    if (!AC) return null;
    ctx = new AC();
    master = ctx.createGain();
    master.gain.value = 0.12;
    master.connect(ctx.destination);
  } catch (_) {
    ctx = null;
  }
  return ctx;
}

function tone({ freq = 440, dur = 0.08, type = 'square', vol = 0.5, attack = 0.005, decay = 0.04, sweep = 0 }) {
  if (!enabled) return;
  const c = ensureCtx();
  if (!c || !master) return;
  if (c.state === 'suspended') c.resume().catch(() => {});

  const osc = c.createOscillator();
  const gain = c.createGain();
  osc.type = type;
  osc.frequency.setValueAtTime(freq, c.currentTime);
  if (sweep) {
    osc.frequency.exponentialRampToValueAtTime(
      Math.max(20, freq + sweep),
      c.currentTime + dur,
    );
  }
  gain.gain.setValueAtTime(0, c.currentTime);
  gain.gain.linearRampToValueAtTime(vol, c.currentTime + attack);
  gain.gain.exponentialRampToValueAtTime(0.0001, c.currentTime + dur + decay);

  osc.connect(gain).connect(master);
  osc.start();
  osc.stop(c.currentTime + dur + decay + 0.02);
}

export const Sound = {
  init() {
    if (typeof window === 'undefined') return;
    try {
      const stored = window.localStorage.getItem('meet:sound');
      if (stored !== null) enabled = stored === '1';
    } catch (_) {}
  },
  isEnabled() { return enabled; },
  setEnabled(v) {
    enabled = !!v;
    try { window.localStorage.setItem('meet:sound', enabled ? '1' : '0'); } catch (_) {}
    if (enabled) {
      ensureCtx();
      if (ctx && ctx.state === 'suspended') ctx.resume().catch(() => {});
    }
  },
  hover() { tone({ freq: 1400, dur: 0.04, type: 'square', vol: 0.18, sweep: -200 }); },
  click() {
    tone({ freq: 880, dur: 0.06, type: 'square', vol: 0.4, sweep: -300 });
    setTimeout(() => tone({ freq: 1320, dur: 0.05, type: 'square', vol: 0.25 }), 30);
  },
  ping() {
    tone({ freq: 1800, dur: 0.05, type: 'sine', vol: 0.35, sweep: 600 });
    setTimeout(() => tone({ freq: 2400, dur: 0.06, type: 'sine', vol: 0.25 }), 40);
  },
  match() {
    const notes = [523, 659, 784, 1046];
    notes.forEach((f, i) => {
      setTimeout(() => tone({ freq: f, dur: 0.1, type: 'square', vol: 0.45 }), i * 90);
    });
  },
  headshot() {
    tone({ freq: 2200, dur: 0.05, type: 'square', vol: 0.5, sweep: -800 });
    setTimeout(() => tone({ freq: 180, dur: 0.08, type: 'sawtooth', vol: 0.3, sweep: -80 }), 20);
  },
  defeat() { tone({ freq: 220, dur: 0.4, type: 'sawtooth', vol: 0.3, sweep: -120 }); },
  round() {
    tone({ freq: 300, dur: 0.18, type: 'sawtooth', vol: 0.25, sweep: 500 });
    setTimeout(() => tone({ freq: 1320, dur: 0.12, type: 'square', vol: 0.3 }), 120);
  },
};

export default Sound;

## 5) ANIMATION & INTERACTION LOGIC

### Smooth scroll (Lenis)
- One `Lenis` instance created in `app/page.js` after the menu is dismissed.
- A `requestAnimationFrame` loop calls `lenis.raf(time)` each frame.
- `MiniMap` calls back into `lenisRef.current.scrollTo(el, { offset: -20, duration: 1.2 })` for smooth in-page navigation.

### Custom crosshair cursor
- `CrosshairCursor.jsx` tracks `mousemove` and stores `(x, y)` in state, positions a fixed-position `<div>` via `transform: translate(-50%, -50%) scale(N)`.
- Scale: `1` default, `1.4` when hovering over `a/button/[role="button"]/[data-interactive]`, `0.7` on `mousedown` (plus a 1px square ring appears briefly).
- Native cursor is killed globally with `cursor: none !important` in CSS, but only on `(hover: hover) and (pointer: fine)` — touch devices keep their native cursor and the component early-returns `null`.
- Uses `mix-blend-difference` so the crosshair stays visible on both light accent and dark backgrounds.

### Main menu state machine
- Three stages in local state: `'menu' | 'settings' | 'matching'`, transitioned via `<AnimatePresence mode="wait">`.
- `PLAY` → set `stage='matching'` → call `Sound.match()` → `setTimeout(onPlay, 1600)` so the parent unmounts the menu.
- Matchmaking screen uses a `.loading-bar` CSS keyframes animation (`width: 0% → 100%` over 1.4s).
- Sound toggle is persisted to `localStorage` via `Sound.setEnabled(v)`.

### Mini-map
- 4 dots positioned absolutely as `%` coordinates inside a 176×176 HUD box.
- Parent (`app/page.js`) keeps `activeSection` updated via an `IntersectionObserver` (threshold 0.35) over the four section IDs.
- Active dot gets `bg-[var(--accent)]` plus the `.pulse-dot` CSS keyframes (expanding `box-shadow`).
- Clicking a dot calls `onNavigate(id)` → Lenis scroll.

### Section reveal animations
- All cards/columns use Framer Motion `whileInView` with `viewport={{ once: true }}` so they animate exactly once.
- Pattern: `initial={{ opacity: 0, y/x: ±20 }} whileInView={{ opacity: 1, y/x: 0 }} transition={{ duration: 0.35-0.5, delay: i * 0.05-0.1 }}`.

### Round-end transition (between sections)
- `RoundTransition.jsx` wraps each section. An `IntersectionObserver` (threshold 0.18) triggers exactly once when the wrapper enters view.
- Plays the configured sound (`round` or `match`) and shows an overlay for 1.1s containing:
  - A 5-rem-tall red bar that `scaleX` 0→1→1→0 with `transformOrigin` switching mid-keyframe from `left` to `right` — produces a wipe-in-then-wipe-out effect.
  - Two thin white sweep lines above/below that scaleX from right and from left respectively on a slight stagger.
- The red bar contains the section title (pixel font) and a checkpoint code (mono).

### Killfeed toasts
- `KillfeedToast.jsx` listens for `scroll` events.
- When the user has scrolled more than 250 px since last check, has a 25% random chance, and 6s have passed since the last fire, it picks an unused FEED entry, shows it for 3.5s.
- When the FEED pool is exhausted, the "fed indices" set is reset.

### Click-to-ping
- `PingEffect.jsx` listens for `click` on `window`. Ignores any click whose target is in an interactive element.
- Adds a `{ id, x, y }` to state, removed after 1.1s.
- Each ping renders as a Framer Motion div animating `scale: 0.2 → 2.5` and `opacity: 1 → 0` over 1s, with a 1.5px center dot.
- Also fires `Sound.ping()`.

### Sound system
- `lib/sound.js` lazy-creates an `AudioContext` on first call, runs everything through a single master `GainNode` at volume 0.12.
- Each method (`hover`, `click`, `ping`, `match`, `headshot`, `defeat`, `round`) builds short oscillator tones with attack/decay envelopes and optional pitch sweep via `exponentialRampToValueAtTime`.
- `SoundProvider.jsx` mounts global capture-phase `mouseover`/`click` listeners. Any `a / button / [role="button"] / [data-interactive]` element gets a hover beep on enter and a click tick on click — except when the click target is inside `[data-no-click-sound]` (used by bird buttons so they only emit `headshot`).
- On/off state is persisted in `localStorage` under `meet:sound`.

### Scrolling system log (bottom-left)
- `SystemLog.jsx` keeps a state array of 6 recent lines (`{ id, text }`).
- On a 1.3s interval, picks a random line from a 22-entry pool (never the most recent one), appends to the array, shifts the oldest off.
- Each line is a Framer Motion `<motion.div layout>`. New lines fade in from `y: 14` to `y: 0` at `opacity: 0.28`. Exiting lines fade out to `y: -14`. The `layout` prop makes the remaining lines slide up naturally.
- Container has a top + bottom CSS `mask-image` gradient so lines fade in/out at the edges.
- z-index is `0` so it sits behind the main menu content but above the menu's dark background.

### Background birds
- `BackgroundBirds.jsx` maintains a state array of alive birds.
- Spawner: on mount, `setTimeout(initial=1200ms)` to spawn the first bird, then a recursive `schedule()` function with random delay 2800–5500ms until at most 3 birds are alive.
- Each bird has a randomized direction (`±1`), starting x just off-screen, target x just past the opposite edge, `y` in the upper ~32% of the viewport, and a speed of 60–140 px/sec.
- The `Bird` subcomponent owns its own `requestAnimationFrame` loop:
  - **Alive**: `x += dir * speed * dt`, every 220ms toggles `frame` 0↔1 to flap the wings. When past the target x, calls `onDespawn(id)` and the component is removed from the array.
  - **Dead** (triggered on click): switches to a different animation in the same loop — `dy = 0.5 * 1600 * t²` (gravity falling), `rot = ±720° * (t / 0.8)` (spin), `opacity = 1 - t/0.8` (fade). After 0.85s calls `onDespawn`.
- Bird elements are `<button data-no-click-sound>` so the global click sound is suppressed; only `Sound.headshot()` plays.
- On kill: increments a `kills` counter (shown as `BIRDS NN` near the bottom) and pushes a `HEADSHOT` or `TARGET ELIMINATED` toast at the click point that animates `y: 0 → -28` then exits.
- Birds are clickable through the menu's transparent middle area thanks to `pointer-events-none` on the menu wrapper and `pointer-events-auto` only on the actual menu block.

### Bird sprites
- Pure SVG with `viewBox="0 0 16 10"`, `shapeRendering="crispEdges"`, and `image-rendering: pixelated`.
- Three frames defined as `[x, y]` rectangle arrays: wings-up, wings-down, dead (with X-eyes).
- `flip` prop applies `scaleX(-1)` when the bird is flying right-to-left.

### Pixel portrait (`AgentBio`) & project skull (`AgentRoster.ProjectSprite`)
- Same SVG pixel-rect technique. Coordinates hand-placed.
- `ProjectSprite` takes a `color` prop so the same sprite renders in red / blue / purple / yellow / green per project.

### Agent detail page
- Static-generated via `generateStaticParams()` — one HTML page per project at build time.
- `params` is awaited (Next.js 15 requirement) in both the page component and `generateMetadata`.
- Animates: hero card slides in on mount; abilities and stack panels animate in on `whileInView`; stat bars animate width from 0 to N% with `whileInView`.

## 6) KNOWN ISSUES

These are the things still rough or worth fixing:

1. **EXIT button is a joke** — currently redirects to `https://www.google.com`. Replace with a proper "thanks for visiting" screen or close-tab attempt.
2. **Placeholder content everywhere** — all 6 projects, all socials (`meet@dev.gg`, `@meet`, etc.), bio, abilities, stats numbers (HP/Armor/Creds, 13-4 score, KDA 4.8) are fake. Swap with real content.
3. **No accessibility for the custom cursor** — `cursor: none !important` is applied globally on non-touch; users with motor difficulties relying on system cursor magnification or high-contrast cursors will lose those. Add a "RESTORE NATIVE CURSOR" setting or respect `prefers-reduced-motion`.
4. **`REDUCED MOTION` setting in Settings is non-functional** — the toggle changes its visual state but doesn't actually disable Framer Motion animations. Wire it into a `MotionConfig` provider or to `useReducedMotion()`.
5. **Birds remain clickable while in their "dead" tumble** — only `stateRef.current.dead` is checked, but the button still receives `pointer-events: auto` during the fall. Add `pointer-events: none` to the button style when `dead === true`.
6. **Birds are scoped to the main menu only** — `BackgroundBirds` is mounted inside `MainMenu`. If you want them everywhere (or in specific sections), pull it up into `page.js`.
7. **Killfeed only fires on `scroll`** — won't fire on mobile if the user uses Lenis-managed scroll which doesn't always emit native scroll events. Consider switching to a Lenis `on('scroll', ...)` subscription.
8. **`generateMetadata` returns the `—` em-dash literally** — fine in titles, but make sure your hosting platform's OG image generation handles unicode.
9. **Mini-map fixed position can overlap the bottom-right of the `MatchSummary` "PLAY AGAIN" CTA on tiny viewports** — consider hiding the mini-map below `sm:` breakpoint with `hidden sm:block`.
10. **Hover/click sounds layer on bird kill** — currently `Sound.click()` is suppressed via `data-no-click-sound`, but `Sound.hover()` still fires when hovering a bird. Add `[data-no-click-sound]` to the hover handler skip-list too if you want a cleaner kill sound.
11. **No `next.config.js`** in this output — the template's default is fine, but if you want static export (`output: 'export'`) you'd need to drop the API route or move it.
12. **`tailwind.config.js` shipped with the original template carries unused shadcn HSL tokens and `tailwindcss-animate`**. I included the minimal version above; you can drop `tailwindcss-animate` and the giant `theme.extend.colors` block entirely.
13. **`prefetch` on every roster `<Link>`** triggers 6 background fetches on the home page. Fine for 6 projects, but if you grow the list, set `prefetch={false}` on cards that aren't immediately above-the-fold.
14. **No `loading.js` or `error.js`** for the `/agents/[slug]` route — invalid slugs trigger Next.js's default 404 page, which doesn't match the HUD theme. Add a themed `not-found.js`.
15. **Press Start 2P has very limited unicode coverage** — characters like `·` (middle dot) and emoji render as `.notdef` boxes when used inside a `.font-pixel` class. Keep pixel-font text strictly to ASCII A-Z 0-9 and punctuation; all middle-dot separators in this codebase are inside `.font-mono` (JetBrains Mono) which renders them correctly.
